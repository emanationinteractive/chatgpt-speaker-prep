⚡ QUICK START (90 seconds):

  1️⃣  EDIT YOUR PROFILE
      Open 2_speaker_prep_prompt.txt (or LOAD_ALL.txt).
      Find SPEAKER PROFILE. Replace the blanks with YOU:
        name=YOUR NAME
        background=YOUR BACKGROUND
        strength_1=YOUR FIRST STRENGTH
        ... etc.
      Save it. This is your character sheet.

  2️⃣  OPEN YOUR AI
      Copilot? ChatGPT? Claude? Gemini? Any of them.
      If Copilot → pick "Smart" mode. Not Think Deeper.

  3️⃣  PASTE 1_START_HERE.txt → Send
      This introduces the tool. AI says "ok ready."

  4️⃣  PASTE YOUR PROMPT → Send
      AI shows startup screen. You're in. 🎮

  5️⃣  TYPE "help"
      See all commands. Try "dump" or "interview" first.

🎮 COMMANDS:

  PREP YOUR TALK:
    dump .......... 🧠 Brain dump. Talk, it saves notecards.
    interview ..... 🎙️ AI interviews YOU. One Q at a time.
    notes ......... 📝 Paste clean notes → notecards.
    parse ......... 💣 Paste messy dump → auto-sorted.
    pitch ......... 🎯 Test your elevator pitch. Scored.
    outline ....... 🗺️ Compile notecards → talk structure.
    review ........ 🔥 Paste draft. Honest feedback.
    comfort ....... 🧘 Save stage comfort techniques.

  LIVE TALK MODE:
    cue ........... 🃏 Map emoji → notecards.
    sheet ......... 🖨️ Print your cue card cheat sheet.
    [emoji] ....... ⚡ Instant notecard. Boom.

  VOICE MODE (earbud whisper):
    "steady" ...... 💚 Confidence anchor in your ear.
    "where am I" .. 🧭 Current outline section.
    "next" ........ ⏭️ What's coming up.
    "stop" ........ 🤫 Go silent.

  SESSION:
    notecards ..... 📋 List all saved notecards.
    show notecards  👀 Read all notecards.
    status ........ 📊 Session stats.
    sos ........... 🆘 Drop format. Just talk to me normally.
    reset ......... 🔄 Back to structured mode.
    help .......... ❓ Show commands.
